# Code bootcamp demo project
## Setup
### Requirements
 * Node 12
 * Visual Studio Code
 
### Init the React frontend 
Following the guide at https://create-react-app.dev/docs/getting-started/

* Run npm install

## Init the Backend
* Run npm install
* Edit package.json and add to scripts
    "start": "node .",
    "start-dev": "nodemon .",

##Starting development
 cd frontend
 npm install
 npm start

 cd backend
 npm install
 npm run start-dev
